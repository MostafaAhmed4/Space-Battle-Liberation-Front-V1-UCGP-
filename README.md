# Space Battle Liberation Front

**Space Battle Liberation Front** is a strategy-based space combat game developed as part of the **2024 Australian STEM Video Game Challenge**. The game followed the storyline of the player commanding a human spacefleet to combat the new AI faction that has turned against the humans.

The project was submitted to the **Open Division (Years 10–12)** of the national competition and was selected as a **Finalist**.

## Australian STEM Video Game Challenge

This project was created for the **Australian STEM Video Game Challenge**, a national competition that encourages students to develop original video games while applying STEM concepts such as programming, mathematics, physics, and game design.

**Result:** Finalist
**Division:** Open (Years 10–12)
**Year:** 2024

## Game Overview


Core gameplay focuses on:

* Strategic positioning in space combat
* Resource and fleet management
* Tactical decision-making during battles
* Progressive escalation of conflict across engagements

## Features
